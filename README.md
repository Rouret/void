  <p align="center">
    By Rouret Lucas
  </p>

Basic template for game development using:
- weppack
- typescript
- nodemon
- express 
- socket.io 

Put your assets in the assets folder and your code in the src folder.

Just tape : 
```bash
npm install
```

And then : 
```bash
npm run start:dev
```

```bash
npm run start:prod
```
